Q2:
A field is a simple variable in a class, while a property provides controlled access (with optional logic) to a value; 
yes, a property can contain logic, like a read-only FullName property that returns FullName.


public class Person
{
    private string firstName;
    private string lastName;

    public Person(string f, string l)
    {
        firstName = f;
        lastName = l;
    }

   public string FullName
    {
        get { return firstName + " " + lastName; }
    }
}
